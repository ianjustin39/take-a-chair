人類大命運 @ Daniel | Thursday, Sep 30, 2021 | 1 minute read | Update at Thursday, Sep 30, 2021

自從看完哈拉瑞的《人類大命運》一書後，這本書就一直是我最愛的讀物之一，也常常名列我最愛推薦給別人的書籍。

近來因緣際會下，我給自己設了一個更艱難的挑戰——用說書的方式來介紹《人類大命運》。

寫「書摘」可說是練習寫作的基本功，我自己也是長時間利用這種文體來磨練自己寫作的能力；但如果想要寫《人類大命運》這種經典書籍的書摘，就會完全是另一種挑戰。

要如何在短短幾千字之內，去蕪存菁同時又符合通篇邏輯脈絡順暢，使人看完後能迅速抓到書本的精髓，一直以來都是寫「書摘」者致力完成的挑戰。

本篇文章與下一篇文章將會是我剖析《人類大命運》的解讀；可以當作簡單的「書摘」來看、也會作為在不久的將來說書的底稿。

沒看過《人類大命運》的人可以把這兩篇文章當作是快速導讀；而看過《人類大命運》的人則可以當作是書本的複習、以及不同人讀書得到不同體悟的相互應證。

//

文章與說書的邏輯將會環繞以下的四個主要框架；前三點將會在本文說明，而最後關於「後自由主義時代」的舉例與剖析則會放在下文：

  1. 引言：人類過去的問題已被解決（饑荒、瘟疫與戰爭），而百尺竿頭更近一步，想要解決的三大新興議題：當「死亡」將走向末日、永遠幸福快樂、智人將演化為「神人」
  2. 過去的三大議題如何出現的？人文主義的興起（泛靈、一神到人文主義）
  3. 貫穿全局的核心：人類說故事的能力（互為主體的想像）、「人類是懸掛在自己編織意義之網上的動物」
  4. 後自由主義時代：科技人文主義與數據主義；當科技與數據比我們更了解自己時，做決策的權力就會從人類自身轉移到演算法手上。 以下正文開始：

//

哈拉瑞編年史：貫穿過去、現在與未來的三大經典

歷史學家哈拉瑞的三本著作中，《人類大歷史》談的是人類如何成為萬物之靈的原因與過程；

《人類大命運》則是從現代出發，點出現有的文化與科技成就是形成於哪些社會脈絡之下，並用可預見的社會脈絡發展來對未來做出預測與想像；

而《21世紀的21堂課》則是專注在當下，談論面臨時代交替、世界動盪的今日該如何自處及深思的種種議題。

三本書所對應到的關鍵字分別是過去、未來與現在；而《人類大命運》對於未來的狂想，更是突破前人所未及。

全書用社會脈絡、思想主義的發展來解釋我們今日社會關切的問題是怎麼產生的；也在同樣邏輯的社會演化論之下，向我們展示未來的種種可能。

//

以古為鏡不只可以知興替，還能洞悉現在、甚至預見未來？

要談論未來，就不得不看過去，這原本就是學歷史的精髓；而身為歷史學家的哈拉瑞，顯然身體力行的實踐了這樣的方法論。

全書開頭以人類過去的三大問題（饑荒、瘟疫與戰爭）在現代已被解決，社會與科學轉而高歌猛進的往三大新興議題來進攻：人類如何達成長生不死、永遠幸福快樂並演化為無所不能的「神人」。

套一句狄更斯在《雙城記》的話：「這是最好的時代，也是最壞的時代。」透過科技的發展，現代社會戰勝了許多老祖宗視為無解的難題； 「史上第一次，因營養過剩而死亡的人數超越營養不良（飢荒）；因年老而死亡的人數超越傳染病死亡（瘟疫）；自殺身亡的人數甚至超越被士兵、恐怖份子和罪犯份子殺害人數的總和（戰爭）。」

到了21世紀早期，平均來說人類死於乾旱、伊波拉病毒或基地組織恐怖攻擊的可能性，還不及死於暴飲暴食麥當勞。

過去困擾人類已久的議題正在逐漸消解，但我們都知道人類並不會滿足於此，於是「想要更多」的心裏讓人們朝著新興的議題前進。

然而這些所謂的「新興議題」，套一句作者的話，只是「過去」的未來；換句話說，這些現在被認定的新興議題，是基於過去三百年的思想和希望，也就是在「人文主義」為指導宗旨下所指向的未來。

然而真正的未來 — — 基於二十一世紀將誕生的新想法、新希望而真正打造的未來（如後文提及的數據主義），可能與「過去的」未來有著截然不同的樣貌。

//

從默默無名到萬物之靈

首先必須先簡單探討在歷史上，「人文主義」這種以人類感受為最高宗旨的意識形態，是如何躍上神壇而主導了近三百年來的世界走向；而人類又是怎麼從跑得不快、也不強壯的孱弱物種，一步步掌權而成為萬物之靈。

大家都知道人類是從「遊牧式」的採集漁獵慢慢轉變到「定居式」的農業社會，而人口與社會發展也隨之擴大；

但較少人認知到的是，從遊牧到定居，其實也影響著社會文化甚至宗教中，人類社會解讀、信仰與對待天地萬物的態度。

採集漁獵時代因為居無定所，人類時常需要靠與自然的共生共存來度過生存的難關，因此那時候的信仰屬於「泛靈信仰」；

老祖宗多半認為萬物皆有靈性，人類只是茫茫萬物中的其中一種，與其他植物、動物甚至無生物地位一般，並沒有什麼特權。

而當人類歷史進展到定居時代的農業社會後，則是因為多數的動植物資源皆是以「人」為中心，萬事萬物皆是人類有意識地養育、栽種而為我所用，自然而然發展出「人類」高於其他物種的心態。

此時的信仰也從「泛靈」進展到「一神」信仰，相信宇宙間有唯一一位至高無上的神祇，而人類則是被選中、被其所愛的子民，有統馭萬物的權利；至此人類與其他物種出現了高下之分。

而到了十六世紀宗教革命後，信仰的舞台則漸漸從「感受全知全能神的意旨」讓步到了「人類個體本身的感受」；

橫空出世的「人文主義」就此萌芽而令人類成為了世界舞台中的唯一主角，並主導了後來世界對於重要議題評斷、經濟科技發展的走向。

總結而論，若快速綜觀人類發展的歷史，可從宗教信仰的態度看出人類是如何一步一步說服自己乃萬物之長：從採集（泛靈信仰）到農業（單一神祇、人類為主）到人文主義（只剩人類）；生活型態的變化影響著人類與其他物種的相處模式，從一開始的眾生平等，到以人為主而天經地義的剝削其他物種生存的權利。

//

「愛妄想」才是人類成功的主因

談完了人類奪權的歷史，接下來書中也探索另一個同樣重要的議題：「人類憑什麼成為萬物之靈」？

人類能有現今的發展眾所皆知靠的是社會的進步與科技的創新，但若把問題再更近一步的向下延伸，到底是什麼樣的力量能讓人類能大規模組織而發展社會制度與科技創新呢？

書中給出了答案，用簡單的一句話來回答就是因為人類有建構並相信「互為主體事實」的能力。

作者在書中提出，除了「主觀事實」與「客觀事實」以外，人類更創造了「互為主體」的第三種事實。

舉例來說，過去各個文明發展中的「神話」，其實就是一種「互為主體的事實」；你相信、我相信，他也相信有神，而當有足夠的人對於同一件事夠相信時，並能以此「事實」為依歸進行大規模的協作，發揮「互為主體事實」的影響力；

而舉實際的例子比如說鈔票，本身的材質就是單純的紙而已，只因為你相信、我也相信鈔票可以當作交易的媒介，交易制度才因而得以幸存甚至到蓬勃發展。

同樣的，也因為人類相信「神祇」、「國家」、或「公司」這樣只存在我們腦海中、卻「互為主體事實」的事物，因而能使人類大規模分工合作而達成其他萬物根本無法比肩的程度；我們透過堅信彼此「想象中」的產物，而能與陌生人無礙的合作。

用賈德戴蒙的名著《槍砲、病菌與鋼鐵》中的字眼來說，由於人類倚靠堅信「互為主體的事實」而擴大了人口社群的規模、擁有與陌生人高度合作的能力後，逐漸發展出定居的農業社會。

而如此社會型態較採集漁獵時代能發展出多餘的糧食，可以養活那些不下田工作的專職人員，進而發展更專精的各行各業技術，例如冶煉金屬的技術，即是“鋼鐵”等科技技術。

而糧食養活更多人，使得社會人口日益增加，不但分工更精細，人與人之間的衝突也會增加，需要發展組織來促進分配效率與統合，因而形成國家等高度分工的精密社會組織；而國家能有效地分配資源，發展軍用征戰品，也就是“槍砲”。

種種人類文明璀璨的背後，單一慣見的事實就是，人類相較於其他萬物是「比較會說故事」且「富有想像力」的物種，能夠相信「互為主體的事實」而為其賣命、受其動員。 正應證了馬克思偉伯的經典名言：「人類是懸掛在自己編織的意義之網上的動物。」

數據主義的未來：演算法會替你找出所有「最佳」決定選項

數據主義認為宇宙是由資料組成，任何現象或實體的價值就在於對資料處理的貢獻；而這樣的社會其實不難想像，現代社會裡提到的「大數據」、「AI」等概念，其實已經擁有「數據主義」所強調概念的雛形。

試想，大數據的本質就是運用大量的資料與演算法，來找出人類無法直接推演出的結論；

比如說在超市中啤酒和尿布放在一起賣的成效最好，原因是因為爸爸們出門買尿布時會順手買一打啤酒。但最一開始時大數據僅是點出了這樣的事實，後來原因是由人類加上去的。

而AI更不用說，Alpha Go用來打敗世界棋王的棋路，很多根本不是人類所能想像，而這些都是AI透過不斷的數據蒐集與演算後無師自通的，完全沒有人類指令的教導。

兩者都是透過數據的不斷蒐集與演算後，得出超越人類判斷的精準分析與預測。

數據主義認為未來的社會裡，人類再也不會是世界運轉的中心。

一開始當然這些人類發明出來的科技仍會應用在「以人為本」等人文主義下關心的議題中，比如加速追求健康、幸福和力量；

數據主義正是透過「承諾會滿足人文主義願望」而得以傳播。而為了實現長生不死、幸福快樂和化身為神的目標，人類就需要處理巨量的資料數據，但這遠遠超出人類大腦的能力，因此只能交給「數據主義」中的演算法。

但等人類慢慢習慣把決策權交給數據後，比如說看病時分析體內的裝置後AI醫生所提出建議的療法、選工作時AI透過全面分析社群帳號行為與LinkedIn上的履歷後來客製化推薦工作、甚至小到晚餐吃什麼而大到該跟誰結婚等等，人類將會如溫水煮青蛙一般逐漸地失去做決策的能力。

畢竟當「數據主義」下的演算法已變得比你還了解自己、給出建議的成功機率也都一目瞭然；與其相信虛無縹緲的直覺而承受失敗的風險，倒不如保守的擁抱演算法所給出的選項。

一旦權力從人類手中交給演算法，人文主義的宏大計劃就可能會慘遭淘汰。

只要我們放棄了以人為中心的世界觀（人文主義），而贊成以資料為中心的世界觀（數據主義），人類的健康和幸福看起來就不再那麼重要了；畢竟都已經出現遠遠超越大腦、更優秀的資料處理模型了，何必再糾結於這麼過時的資料處理機器呢？

在這樣的未來裡，人類有可能從設計者降成晶片、再降成數據資料，甚至成為駭客任務電影中單純提供能量的電池，最後在資料數據的洪流中溶解分散，如同滾滾洪流中的一搓泥沙。

於是，數據主義對人類造成的威脅，正如人類對其他動物造成的威脅；一旦人類對網路不再能提供重要的功能，就會發現自己到頭來也不是造物的巔峰。

我們自己設下的標準，會讓我們也走上長毛象和白鯨豚遭到滅絕的死路。世界的權力中樞從「萬物」、「單一神祇」、「人類」到「數據」後，多數人類的存在將會無足輕重，而就此消失在歷史發展的舞台中。

//

《人類大命運》：洞悉多重未來的可能

本書最重要的目的，是要透過預測未來科技進步後，人類所會發展出的新理論價值；而這樣的思考邏輯與視野高度，是用現在的思維所無法完全體認的。

這也是後文沒有再針對新興的人類三大議題多作著墨，乃是因為在預見的未來中，「人類」的需求不再會是社會發展的重點。

若仍拘泥於書中開頭所提到的三個現今看來重要的問題，而不能體會作者嘗試用另一個維度的未來邏輯來解釋社會，將會十分可惜。

這就像是為什麼生活在中古世紀的人，無法理解「社會主義」為何探討的議題是階級對立，因為從他的視角來說，階級根本不該有流動的可能。

不能跳脫自己當下維度的思考邏輯，而以現有的社會建構性來審視本書，只會得到盲人摸象的片面結論。

科技的發展並不會帶來決定性的結果，端看應用者如何處理；人工智慧和生物科技的興起肯定會改變社會，但並不代表只會有一種結局。

《人類大命運》書中講到一切的情境都只是可能性而非預言，主旨是要帶領讀者體悟一種按照歷史脈絡思考未來的能力，並擴展視野而思考未來世界走向的各種可能性。

歷史總是無情地向前，天地也以萬物為芻狗，因此身處在關鍵轉變時代的我們，若不儘早思考某些會永遠影響未來社會走向的議題，那人類終將不可避免地被時代的巨輪壓垮而淘汰在歷史的洪流之中。

未來的社會仍有多種可能，端看身處在時代交會的我們，如何決定世界未來的走向。
